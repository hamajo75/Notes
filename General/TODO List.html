TODO Gerhardt - Individuum und Menschheit: Eine Philosophie der Demokratie

I. Anfänge von Menschheit und Demokratie
1. Das Parallelogramm von Mensch und Politik

Platon - Politeia: Der Staat ist ein idealisierter Mensch. Oder umgekehrt: Der Staat bildet für den Menschen ein Modell, hinsichtlich seiner Eigenschaften und Tugenden. An diesem kann sich der individuelle Mensch orientieren. Staat und Mensch sind sich ähnlich. In diesem Sinne spricht man auch von Staatskörper und seinen Organen.

Rede von Menenius Agrippa (nach Titus Livius): Er bringt die revoltierenden Truppen Roms zur Rückkehr durch einen Vergleich des Staates mit dem menschlichen Körper. Die Gliedmaßen revoltieren gegen den Magen und verweigern ihm die Nahrung, sehen aber am Ende ein, dass nur alle gemeinsam überleben können.

Die gesamte Konstruktion des Politischen baut auf den zwei Pfeilern: Individualität und Allgemeinheit. Darauf aufbauend entsteht vor ca. 2500 Jahren ein Begriff der Menschheit angelehnt an einen politische Verfassung, die Freiheit und Gleichheit des Individuums garantiert.

2. Menschheit als Verständigungsgemeinschaft

Menschen und Götter: Die Menschen sehen sich, den Göttern unterworfen, in einer natürlichen Herrschaftsordnung. Sie bilden eine Verständigungsgemeinschaft, die sie zu gewissen Handlungen verpflichtet. Sie haben gemeinsame Vorstellungen und Überzeugungen, die flexibel sind und sich verändern. Menschen lernen voneinander und sind immer aufeinander bezogen.

Zum gegenseitigen Verstehen gesellen sich auch Irrtum, Täuschung und Lüge.

Wo es stabile Herrschaftsordnungen gibt, werden einzelne Menschen herausgehoben und als Individuen bewundert. Individuelle Leistungen definieren Tugenden. Auch heben die Götter den Menschen in eine exponierte Stellung. Es kommt zur Herausbildung des Begriffs der Individualität.

Was immer Menschen gemeinsam tun, es basiert auf Abstraktionen, die alle beteiligten verstehen müssen.

3. Der Anteil der ersten Philosophen

Xenophanes, Heraklit, Anaxagoras denken über die Gegriffe Gott, Mensch, Natur, Geist nach. Sie thematisieren Individualität und Universalität.

4. Der geschichtliche Vorlauf der Demokratie

Herodot und Thukydides erzählen uns vom Peloponeschischen Krieg. Der günstige Ausgang der Auseinandersetzung mit den Persern ermöglicht es Athen unter der Führung des Perikles eine Demokratie zu werden. In den Historien von Herodot findet sich die Beschreibung einer Debatte am persischen Königshof über die beste Staatsform. Diskutiert werden

  1. Monarchie (d. Herrschaft eines einzelnen), König Darius
  2. Aristokratie (d. Herrschaft der besten), ein gewisser Otanes
  3. Demokratie (d. Herrschaft des Volkes), ein gewisser Megabyzos

Auf die Demokratie ist zu setzten, weil nur sie, durch ihre vielen

  • Ämter und
  • Zuständigkeiten des Volkes
  • Verfahren

die Kontrolle der Regierenden gewährleistet.

Auf der Gesamtheit des Volkes ruht der ganze Staat.

Diese Idee trifft auf einhellige Ablehnung, aufgrund der Inkompetenz des Volkes.

Die Pointe des Otanes war aber eine institutionelle: Die Kompetenz der demokratischen Institutionen beruht nicht auf der Kompetenz des Einzelnen.

Die Rede des Otanes spricht auch über die Gleichheit vor dem Gesetz. Eine Bedingung für diese ist die Freiheit des Bürgers. Freiheit und Gesetz gehören zusammen. Freiheit und Gleichheit sind nicht zu trennen.

Idee: Einem guten Gesetz folgt der Bürger freiwillig.

Diese Gleichheit steht gegen einen Alleinherrscher. Er wird sich nie als gleichberechtigter Bürger verstehen.

Solons Verfassung: Reformen in Athen (595 v.C.)

  • gleicher Rechtsanspruch für alle Bürger
  • Einfluss auf die Besetzung der Gerichte
  • Einfluss auf die Gesetzgebung
  • Reglungen bzgl. Schulden
    • 480 v.C.: Eroberung Athens durch die Perser.
5. Die weltpolitische Innovation in Athen

Im ersten Jahr des peloponnesischen Krieges 431 vor Christus haben wir Athener siegreich gekämpft. Thematisiert wird die Rede des Perikles für die Gefallenen (Epitaphios). Geschichtsschreiber Thukydides überliefert uns die Rede. unter dem Eindruck des Totalitarismus des 20 Jahrhunderts gab es Autoren denen die Rede des Perikles den Mut und die Kraft gegeben hat, trotz allem an die Zukunft der Demokratie zu glauben (Popper übersetzt sie, im neuseeländischen Exil ins Englische).

  • Freie Entfaltung der Bürger.
  • Demokratie als Bildungsgemeinschaft.
  • Didaktisches Mittel ist das gute Beispiel.
  • Jeder kümmert sich nicht nur um das private sondern auch um die öffentlichen allgemeinen Dinge.
  • Jeder hat ein Interesse sein eigenes Leben mit dem alle anderen zu verbinden.
  • Demokratie ist nie reine Rechtsordnung, sondern eine Lebensform.
  • Rechtsgleichheit, keine Privilegien aus Herkunft und Besitz.
  • Anerkennung der Meinungsverschiedenheit als wesentlicher Element einer Gesellschaft (Pluralität).

Erzähltheorie allgemien
  • https://de.wikipedia.org/wiki/Erz%C3%A4hltheorie

Gegenstand: Alle Arten erzählender Texte bzw. erzählender Textabschnitte.

Thematische vs. modale Narratologie:

  • Was wird erzählt (Ereigniskette)?
  • Wie wird erzählt (Stilmittel, etc.)?
Erzählperspektiven
  • Der allwissende Erzähler (auktorialer Erzähler) Susanne S. ging die Straße entlang. Sie konnte noch nicht ahnen, was noch passieren würde.
  • Neutraler Erzähler: Perspektive eines unsichtbaren Beobachters (wie Kamera). Susanne S. ging die Staße entlang. Sie hatte ein rotes Kleid an.
  • Erzählung aus der Perspektive einer bestimmten Figur (Personale Erähler). Sie ging die Staße entlang und dachte sich….
  • Ich-Erzähler (Innenperspektive) Ich ging die Straße entlang und dachte mir…
    • erzählendes Ich vs. erlebendes Ich: Ich erzähle euch jetzt was: Vor ein paar Jahren…

2 Die Erzählung
2.1 Von der erzählenden Ereignisverknüpfung zur Erzählung
  • zentraler Begriff: narrative (erzählende) Ereignisverknüpfung
    • min. 2 Ereignisse werden chronologisch geordnet und in min. einer weiteren Weise sinnvoll verknüpft dargestellt.

also: Chronologisch geordnete und sinnvoll verknüpfte Ereignisdarstellung.

  • Sinnvolle Verknüpfungen
    • kausal: In Folge der Hungersnot ging er nach Amerika.
    • Wünsche, Hoffnungen, Absichten, Handlungsmotive
    • Gründe (um-zu-Relation)
    • evaluativ verknüpft: ein Ereignis stellt sich auf Grund eines anderen als bedauerlich, erfreulich etc. heraus.

Kompetenzen des Lesers

  • Ergänzung von Lücken
  • chronologische, kausale Ordnung von Ereignissen
  • Annahmen von: Erzählstimme, Perspektive, Absichten, etc.
  • 3 Bedingungen für das Vorliegen einer Erzählung
    1. Bezugnahme auf die Ereignisse
    2. Spannugnsbogen
    3. Tellability

    ad 1) Relationen zu Personen

    • Welche Beziehung haben die Ereginisse zu Personen (Hoffnugnen, Bedürfnisse, Meinungen).
    • Aus welchem Grund sind die Ereignisse wichtig?

    ad 2) Abgeschlossenheit

    • Abgeschlossenheit (Anfang, Mitte, Ende - vgl. Aristoteles Poetik)
    • kognitiver Aspekt: Frühere Ereignisse erzeugen eine bestimmte Erwartung. Spätere Ereignisse befriedigen diese Erwartung (Abschluss d. Spannungsbogens).
    • affektiver Aspekt: Folge von Gefühlen, die der Leser durchlebt kommt zu einem Abschluss.

    ad 3) tellability oder narrative point

    • Pointe, d. Witz an der Sache (Warum ist die Erzählung nicht banal?)
    • kann sein: Unterhaltungswert, Ästhetik, Informationsgehalt
2.2 Fiktionale Erzählungen
  • Der Leser ist aufgefordert sich etwas vorzustellen (Autorenwelt, Leserwelt).

3 (unter Umständen dramatisch verschiedene) Welten:

  1. Welt des Authors
  2. Welt des Lesers
  3. die abstrakte Welt der Erzählung

Vorstellungen

  • bewusst, sie existieren so lange man sich etwas vorstellt
  • willentlich, sie können (zumindest teilweise) gesteuert werden
  • Simulationstheorie: Vorstellungen sind Realitätssimulationen

Wahrheit und Fiktionalität

  • Eine Erzählung wird nicht dadurch fiktional, dass sie falsche Sätze enthält. Eine Lügengeschichte kann durchaus außerhalb eines fiktionalen Kontextes erzählt werden. Das ist gerade wesentlich beim Lügen.
  • Ein Satz wird dadurch fiktional, dass er in einer Fiktionalitätssituation geäußert wird.

Fiktionalität vs. Fiktivität

  • Fiktional sind Ausdrücke, Sätze, etc., die in einem fiktiven Kontext geäußert werden.
  • Fiktiv ist das, von was die Erzählung handelt. Die Dinge, die man sich vorstellt.
  • Man kann sich natürlich über die Fiktionalität einer Erzählung irren.
2.2.1 Die Rolle des fiktiven Erzählers in fiktionalen Erzählungen

Autor vs. Erzähler

  • Autor produziert den Text.
  • Erzähler: fiktive Instanz, die zur Vorstellung auffordert, der Text würde von jemandem erzählt.

  • Der Erzähler kann Teil der fiktiven Welt sein oder auch nicht.

    Es muss keinen fiktiven Erzähler geben: Die Katze lag auf dem Teppich und das Sonnenlicht fiel durch das Fenster auf sie. Die Szene glich einem Stillleben.

2 kinds of errors:

  1. recoverable errors
    • file not found
    • unknown message type
    • convert string to int
  2. non recoverable errors
    • out of range
    • null ptr dereference
    • segmentation fault

In case 2 we can really just stop the program and give as much good information as needed to find out what went wrong.

happy path, error path

Error handling techniques:

Error codes

  • return bool
  • return int error code
  • return std::optional - that's like an advanced bool
  • return std::variant - very noisy

  • return std::expected

    Notes:

  • Happy code and error code are mixed up together. They're not really well separated.
  • Functional composition is difficult.

Exceptions

TODO C++ Template Metaprogramming

  • https://www.linux-magazin.de/ausgaben/2011/01/c/

Calculate types and values at compile time.

History

  • Erwin Unruh (1994) - with templates you can calculate values at compile time.
  • Became popular with Modern C++ Design (Alexandrescu).

Examples

Calculate a value at compile time
// Define a template function that calculates the factorial of a given number
template <int N>
int factorial()
{
    // Use recursion to calculate the factorial of N
    return N * factorial<N - 1>();
}

// Define a specialization of the factorial function for the base case (N = 1)
template <>
int factorial<1>()
{
    return 1;
}

int main()
{
    // Calculate the factorial of 5 at compile time
    int result = factorial<5>();

    // Print the result
    std::cout << "5! = " << result << std::endl;

    return 0;
}
Change types at compile time
#include <iostream>
#include <map>
template <typename T>
struct RemoveConst{
    typedef T type;
};
template <typename T>
struct RemoveConst<const T>{
    typedef T type;
};
template <typename T>
void checkConst(){
    if (std::is_const<T>::value == true  ) {
        std::cout << "const " << std::endl;
    }
    else {
        std::cout << "not const" << std::endl;
    }
}
int main(){
  std::cout << "int : "; checkConst<int>();
  std::cout << "const int: "; checkConst<const int>();
  std::cout << "RemoveConst<int>::type > "; checkConst< RemoveConst<int>::type >();
  std::cout << "RemoveConst<const int>::type > : "; checkConst< RemoveConst<const int>::type >();
  typedef RemoveConst<const int>::type NotConstFromConstInt;
  std::cout << "NotConstFromConstInt : "; checkConst< NotConstFromConstInt >();
